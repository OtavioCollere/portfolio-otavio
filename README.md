# portfolio-otavio

Portfólio pessoal desenvolvido com Next.js, TypeScript e Tailwind CSS.

## 🔗 Demo ao vivo

[portfolio-otavio-orpin.vercel.app](https://portfolio-otavio-orpin.vercel.app)

## Stack

- **Framework:** Next.js 14 (App Router)
- - **Linguagem:** TypeScript
  - - **Estilização:** Tailwind CSS
    - - **Deploy:** Vercel
     
      - ## Sobre o projeto
     
      - Portfólio desenvolvido para apresentar minha experiência profissional, stack técnica e projetos. Inclui seções de experiência, projetos em destaque, competências e informações de contato.
     
      - ## Como rodar localmente
     
      - ```bash
        # Instalar dependências
        npm install

        # Iniciar servidor de desenvolvimento
        npm run dev
        ```

        Acesse [http://localhost:3000](http://localhost:3000).
